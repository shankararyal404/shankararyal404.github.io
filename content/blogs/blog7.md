## User Documentation

### Installation

```bash
git clone https://github.com/MrShankarAryal/ShadowStrike.git
cd ShadowStrike
pip install -r requirements.txt
python gui.py
```

### 1.1. Reconnaissance

**Start Recon:**
- Enter the Target IP
- Click Start Recon
- Review results in log box

### 1.2. Attack Modes

**Choose Attack Mode:**
- **SYN Flood**: TCP connection overload
- **HTTP DDoS**: Web server stress testing
- **Stealth Probe**: Low-profile scanning
