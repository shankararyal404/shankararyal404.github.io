ShadowStrike is designed exclusively for educational purposes and ethical security testing. Use ShadowStrike only in authorized testing environments, maintain proper documentation, and follow responsible disclosure practices.

Installation

git clone https://github.com/MrShankarAryal/ShadowStrike.git
cd ShadowStrike
pip install -r requirements.txt
python gui.py

1.1. Reconnaissance

Start Recon:

  • Enter the Target IP
  • Click Start Recon
  • Review results in log box
