# Shankar Aryal Portfolio Website

This is the personal portfolio and blog website for Shankar Aryal.

## 📁 Project Structure

*   **`public/`**: The frontend source code (HTML, CSS, JS). Serving root.
*   **`functions/`**: Cloud Functions for Firebase (Backend logic).
*   **`admin/`**: Admin panel for content management.
*   **`config/`**: Configuration files.
*   **`docs/`**: Documentation.

## 🚀 Getting Started

1.  **Install dependencies** (for backend):
    ```bash
    cd functions
    npm install
    ```

2.  **Run locally**:
    You can serve the `public` directory using any static file server.
    ```bash
    python -m http.server 8000
    # Visit http://localhost:8000/public
    ```

## 🛠 Tech Stack

*   **Frontend**: HTML5, Vanilla CSS3 (Modular), JavaScript (ES6 Modules).
*   **Backend**: Node.js (Firebase Functions).
*   **Database**: Firestore.
*   **Storage**: Firebase Storage.
